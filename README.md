# 📸 以图搜同款

上传商品图片，一键在 **1688 / 淘宝 / 速卖通** 搜索相似款式。

## 功能

| 平台 | 状态 | 说明 |
|:---|:----|:----|
| 🟢 **1688 图搜** | ✅ 自动 | 上传图片自动跳转1688以图搜源，最准 |
| 🟢 **淘宝 图搜** | ✅ 自动 | 上传图片自动跳转淘宝拍立淘 |
| 🟡 **速卖通** | ⚠️ 手动 | 速卖通已关闭站外图搜API，点击后打开速卖通主页 → 搜索框相机图标手动上传 |
| 📋 搜索历史 | ✅ | 本地保存最近搜索记录 |
| 🔗 图片链接 | ✅ | 支持粘贴网络图片链接直接搜索 |

## 为什么用1688？

做跨境的都知道：**速卖通上的货，80%以上来自1688**。用1688图搜不仅能找到同款，还能看到批发价。本工具优先推荐1688图搜。

## 使用

直接打开 `index.html` 即可使用，无需部署。

或在线访问：[GitHub Pages](https://aaron15295066509-byte.github.io/ali-image-search/)

## 技术栈

纯前端 HTML + CSS + JS，无后端依赖。图片通过 sm.ms 免费图床暂存。
