# 📸 以图搜同款 - 全平台

上传商品图片，一键在 **6大电商平台** 搜索相似款式。

## 支持的平台

| 平台 | 类型 | 说明 |
|:---|:----|:----|
| 🟢 **1688** | 🤖 自动 | 上传图片自动跳转1688以图搜源，跨境首选 |
| 🟢 **淘宝** | 🤖 自动 | 上传图片自动跳转淘宝拍立淘 |
| 🟢 **Temu** | 🤖 自动 | 上传图片自动跳转Temu图搜 |
| 🟡 **AliExpress** | 👆 手动 | 打开速卖通主页 → 搜索框📷相机上传 |
| 🟡 **Amazon** | 👆 手动 | 打开亚马逊主页 → 搜索框📷相机上传 |
| 🟡 **SHEIN** | 👆 手动 | 打开SHEIN主页 → 搜索框📷相机上传 |

## 使用

1. 拖拽或点击上传商品图片
2. 点击目标平台按钮
3. **自动平台**：直接跳转搜索结果页
4. **手动平台**：打开主页后，在搜索框点击相机图标上传
